    Blind Code is an opportunity to check the smartness of your brain!! Ever thought of writing the code with MONITOR OFF?!!!.. Sounds challenging?!! This time RENAISSANCE 2K19 brings you the most intellectual event where the participants will have to code with MONITORS switched OFF. So unlock your zest to register yourself for the most enthralling event.


  • 1.Individual Event
  • 2.The compiler provided will be TURBO C
  • 4.The event will be 2-3 hour long.
  • 5.There will be a total of two rounds.
  • First Round: -A simple code with syntax error will be given on paper.
  • Participants have to correct the errors on paper and type the code with MONITOR SWITCHED OFF.
  • Second Round: -Only problem statement will be provided. (Only the complexity of the program and time limit will be increased).
  • 6.The team may consist of a maximum of 1 participant.
  • 7.Simple logical program based on data structure and algorithm will be given to The participants.
  • 8.The participants are required to write the proper executable code in C/C++ Language which is given in the first round with errors and participant have to solve it and execute the code without power on the screen and come up with the correct output Within the pre-decided time limits.
  • 9.If any of the participants are not able to solve the problems within the pre-Decided time limits then the decision of selecting the participants will be taken based on the best code(code with minimum errors wins).
  • 10.Time limit for the first round will be 20 minutes and for the second round, it will be 30 minutes.
  • 11.Problems will be assigned by the organizers of an event.
  • 12.The participants will be provided with the desired work stations. The Workstations does not have any net connections and neither any individual Phones.
  • 13.If any team is found copying a code that has already been debugged/Switching on the screen, then he/she will be disqualified on the spot.
  • 14.Rules are subject to change prior to the briefing session.
  • 15.The decision of judges will be final.
