    Procode

    It will be a 2 - 3 hour Competitive programming contest hosted on Hacker rank (https://www.hackerrank.com/). The Contest consists of 5 - 6 problems based on Data structures and Algorithms, Mathematics. Students have to code the solution in any programming language of their choice. In this contest problems are designed in such a way along with the ability to test data structures and algorithms, the candidate is also checked with fundamentals of mathematics, logical aptitude, time and space complexities knowledge. If a candidate has command on any other language he/she can do coding in that language. We are mainly concerned with the output under our given test cases and under given time constraints. The competition will be like TCS Code Vita and prepare students for contests like TCS Code Vita.


  • 1. It is an INDIVIDUAL contest (i.e. Team Size= 1 person/team).
  • 2. Teams will be ranked on the basis of SCORE gained.
  • 3. Ties will be broken by the total time for each user in ascending order of time.
  • 4. Total time is the sum of the time consumed for each problem solved. The time consumed for a solved problem is the time elapsed from the beginning of the contest to the submission of the first accepted run plus some penalty minutes for every previously rejected run for that problem (Wrong answer, Time limit exceeded, Runtime Error, Compilation Error). There is no time consumed for a problem that is not solved.
